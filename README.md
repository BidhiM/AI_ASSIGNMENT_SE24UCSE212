📌 Assignment 1: MISSIONARY CANNIBAL Problem  
File: missionary_cannibal.py  
This assignment implements the classic missionary-cannibal problem, a well-known Artificial Intelligence search problem.  
The objective is to control the number of missionaries and cannibals by applying operations such as:  
number of missionaries should always be greater than number of cannibals on either side  
the boat can carry a maximum of 2 people  

📌 Assignment 2: AQI Reflex Agent (Multi-Pollutant System)  
File: aqi_reflex_agent_multi_pollutant.py  
Environment File: environment.txt  
This assignment implements a Reflex Agent that monitors multiple air pollutants and determines air quality conditions based on environmental input data.  
The system reads pollutant data from environment.txt and makes decisions according to predefined rules.  
The environment.txt file acts as the input environment containing pollutant values such as:  
PM2.5  
PM10  
CO  
NO2  
SO2  
O3  
Make sure environment.txt is in the same directory.
